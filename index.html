Projects

Multi-Modal 3D Hand Pose Estimation

The thesis focuses on 3D multi-modal hand pose estimation using acceleration data from a sensor-equipped glove and video input. It integrates MediaPipe overlays for visual hand tracking, combining sensor data and computer vision techniques. This approach enhances pose accuracy and robustness for applications in VR, robotics, and human-computer interaction. Technologies Used: Machine learning, Mediapipe, Python, CNN.

CNN Binary Image classifier

Multiclass Image classifier along with GAN

Multiclass Image classifier along with GAN This is personal project built as a hobby. Implemented model using tensorflow library for Multiclass classification of images. Basic GAN has been built which works on MNIST and MNIST fashion dataset and generates new images.
